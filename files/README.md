# Example Files

Add your files here. This folder will be served as the root of your file browser.

## Folder Structure Example

```
files/
├── images/
│   ├── photo1.jpg
│   └── photo2.png
├── documents/
│   └── readme.md
├── MyFont.font/          # Font folder (special handling)
│   ├── MyFont-Regular.otf
│   └── MyFont-Bold.otf
└── music/
    └── song.mp3
```

## Font Folders

Any folder ending in `.font` gets special treatment:
- Click opens a font preview modal
- Slider to switch between font styles
- Download individual fonts or zip all
